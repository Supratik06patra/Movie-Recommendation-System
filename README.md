# Movie Recommendation System

## Project Overview

This project is a Movie Recommendation System built using Python and Machine Learning techniques. The system recommends similar movies based on movie genres and descriptions.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Google Colab

## Features

* Movie Recommendation
* Content-Based Filtering
* Similarity Analysis
* Fast Recommendation Engine

## Dataset

TMDB 5000 Movies Dataset

## Model Used

CountVectorizer + Cosine Similarity
